Sécurité Web – 404CTF

J’ai résolu plusieurs challenges portant sur la sécurité des applications web, qui m’ont permis de pratiquer l’exploitation de vulnérabilités courantes dans un environnement contrôlé.

  • JWT falsification : manipulation de jetons d’authentification JSON Web Token pour contourner les mécanismes de vérification d’identité
  • Path Traversal : exploitation d’une faille de navigation dans l’arborescence du système de fichiers

Ces challenges m'ont permis de renforcer ma compréhension du fonctionnement des applications web côté serveur, des erreurs de configuration courantes et de l’impact des mauvaises pratiques de sécurité dans la gestion des données et des accès.

Analyse Forensique – 404CTF

Résolution d'un challenge en analyse forensique portant sur l’exfiltration de données via un périphérique USB à partir d’un fichier de capture réseau au format .pcapng.

  • Ouverture du fichier avec Wireshark et application d’un filtre usb
  • Identification de paquets URB Bulk OUT via Leftover Capture Data
  • Extraction des données au format binaire (.bin)
  • Conversion du contenu binaire en texte pour révéler le flag

Sécurité Matérielle – 404CTF

Dans ce challenge, j’ai dû reconstruire un signal audio à partir d’un fichier IQ (chall.iq) contenant des données complexes issues d’une transformée de Fourier.

  • Chargement des données IQ avec numpy
  • Application de l’IFFT
  • Normalisation, conversion int16 et export WAV
  • Écoute du signal pour identifier le flag

Bien que guidé à 70% dans la démarche, j’ai pris soin de comprendre chaque étape.
